# mini-http-server(C#)
is being developed.

An educational HTTP server written in C# from scratch without ASP.NET.

The project is being developed step by step to understand how HTTP works internally.

Later this project will become the basis for **ZeroVault** — a browser-based zero-knowledge encrypted vault.

---

## Features

- TCP server using TcpListener
- Manual HTTP request parsing
- Static file serving
- Routing
- MIME type detection
- GET support
- HEAD support
- OPTIONS support
- Access logging
- Error logging
- Path Traversal protection
- Security HTTP headers

---

## Current project status

The server can:

- serve HTML pages
- serve CSS files
- detect MIME types
- return HTTP status codes
- write access/error logs
- block Path Traversal attacks
